# 鼠标右键截屏工具

这是一个轻量的 Windows 右键菜单工具。安装后，在桌面或文件夹空白处点击鼠标右键，即可选择“截屏”，调用 Windows 自带的截图工具。

## 安装

双击 `安装截屏右键菜单.cmd`，看到“安装成功”即可。

不需要管理员权限，仅对当前 Windows 用户生效。

## 使用

1. 在桌面或文件夹空白处点击鼠标右键。
2. 点击“截屏”。
3. 框选要截取的区域。

在 Windows 11 中，如果“截屏”没有直接出现，请点击右键菜单底部的“显示更多选项”。

## 卸载

双击 `卸载截屏右键菜单.cmd` 即可完整移除菜单项。

## 系统要求

- Windows 10 或 Windows 11
- 系统自带“截图工具”
