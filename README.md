# 鼠标右键截屏工具

这是一个轻量的 Windows 全局右键截屏工具。安装后，在桌面、网页或其他软件中按住 `Ctrl` 并点击鼠标右键，即可选择“截屏”，调用 Windows 自带的截图工具。普通右键菜单不会受到影响。

## 安装

双击 `安装截屏右键菜单.cmd`，看到“安装成功”即可。

不需要管理员权限，仅对当前 Windows 用户生效。

## 使用

1. 在任意界面按住 `Ctrl` 并点击鼠标右键。
2. 点击“截屏”。
3. 框选要截取的区域。

工具安装后会在后台运行，并随 Windows 登录自动启动。也可以双击任务栏通知区域中的截图图标直接截屏。

## 卸载

双击 `卸载截屏右键菜单.cmd` 即可完整移除菜单项。

## 系统要求

- Windows 10 或 Windows 11
- 系统自带“截图工具”
